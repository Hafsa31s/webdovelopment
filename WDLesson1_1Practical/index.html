Web Technologies

HTML

HTML stands for Hypertext Markup Language,
and it is the standard markup language for creating web pages
and structuring web content.It is a coding language to display
content on webpage.

CSS

CSS stands for Cascading Style Sheets.
CSS describes how HTML elements are to be displayed on screen, paper,
or in other media. CSS saves a lot of work. It can control the
layout of multiple web pages all at once.

JavaScript

JavaScript is a dynamic programming language
that's used for web development, in web applications,
for game development, and lots more. It allows you to implement dynamic
features on web pages that cannot be done
with only HTML and CSS.
